### 1.2 名词定义

| 术语 | 说明 |
| --- | --- |
| 画册（BabyBook） | 由多张宝宝照片按模板排版生成的电子/可打印相册 |
| 模板（Template） | 决定画册整体风格的版式方案，包括封面、内页、背景、字体、装饰元素等 |
| 页面（Page） | 画册中的单页，由图片位、文字位、装饰位组成 |
| 版式（Layout） | 单页内图文的排列组合方式 |

## 二、项目背景与目标

### 2.1 项目背景

父母在育儿过程中会累积大量宝宝照片，散落在手机相册、微信、云盘中，缺少系统化的整理和纪念方式。市面上传统相册制作门槛高（需要手动排版、设计），成年人难以抽出时间制作；而简单的电子相册又缺乏仪式感与美观度。

随着 AIGC 能力成熟，我们可以通过智能排版 + 高质量模板，让用户只需上传照片，即可一键生成精美画册，兼顾"轻量操作"与"高质量成品"。

### 2.2 产品目标

- **核心目标**：让用户最少操作成本下，把宝宝照片生成为一本精美的电子画册。
- **业务目标**：
  - 完成画册的用户转化率 ≥ 60%
  - 画册分享率 ≥ 30%
  - 模板人均使用数 ≥ 2
  - 付费转化率（付费模板/导出高清/打印）≥ 8%

### 2.3 目标用户

| 用户画像 | 核心场景 | 核心诉求 |
| --- | --- | --- |
| 新手妈妈（0-1 岁宝宝） | 记录百日、周岁等重要节点 | 操作简单、模板温馨、成品精美 |
| 宝爸宝妈（1-6 岁宝宝） | 生日、节日、旅行纪念 | 多主题模板、可分享 |
| 祖辈 | 给孙辈做纪念册送礼 | 易上手、可打印成实体书 |

## 三、产品范围

### 3.1 MVP 版本范围（V1.0）

- ✅ 用户上传宝宝照片（本地相册 / 云相册）
- ✅ 提供 10+ 种精美模板，覆盖不同主题风格
- ✅ 基于所选模板自动排版生成画册
- ✅ 画册预览、翻页浏览
- ✅ 画册保存到"我的画册"
- ✅ 画册分享（H5 链接、图片长图）

### 3.2 非 MVP 范围（后续迭代）

- ⏳ AI 智能挑图（自动从大相册中挑出宝宝照片与高质量照片）
- ⏳ AI 成长文案生成（根据照片时间、场景自动生成配文）
- ⏳ 用户自定义模板（编辑文字、更换背景、调整版式）
- ⏳ 实体画册打印与邮寄
- ⏳ 多人协作共建（爸妈、长辈共同添加照片）
- ⏳ 宝宝成长时间轴（按月/年自动归档）

## 四、核心功能需求

### 4.1 功能架构图

```
宝宝画册 App
├── 画册创建
│   ├── 上传照片
│   ├── 选择模板
│   ├── 智能排版
│   └── 预览生成
├── 画册管理
│   ├── 我的画册列表
│   ├── 画册详情/翻阅
│   ├── 删除/重命名
│   └── 分享/导出
└── 模板中心
    ├── 模板分类浏览
    ├── 模板详情预览
    └── 收藏模板
```

### 4.2 核心流程：画册生成主流程

```
[进入首页] → [点击"创建画册"] → [选择照片] → [选择模板]
   → [生成中（Loading）] → [画册预览] → [保存 / 分享 / 导出]
```

---

### 4.3 功能详述

#### F1：上传照片

**功能描述**：用户从本地相册或云端相册中选取宝宝照片，作为画册内容来源。

**功能要求**：

| 项 | 要求 |
| --- | --- |
| 照片数量 | 最少 6 张，最多 60 张（单册） |
| 照片格式 | JPG、PNG、HEIC，单张 ≤ 20MB |
| 照片来源 | 本地相册、相机拍摄；后续支持微信、云盘导入 |
| 排序方式 | 默认按拍摄时间正序；用户可拖拽调整顺序 |
| 照片预处理 | 自动识别并提示以下问题：模糊、重复、过曝、无人物；由用户决定是否剔除 |
| 隐私保护 | 仅本地选图授权，照片上传走 HTTPS；用户可随时删除 |

**异常处理**：

- 不足 6 张：按钮置灰并提示"至少选择 6 张照片"
- 超出 60 张：保留前 60 张并提示"已超出上限，保留前 60 张"
- 格式不支持：自动过滤并提示

---

#### F2：模板选择（★核心）

**功能描述**：提供多种风格模板，用户选择后决定整本画册的视觉风格。

**模板分类（至少 5 大类，每类 2-3 款，MVP 共 ≥ 10 款）**：

| 分类 | 风格关键词 | 适用场景 |
| --- | --- | --- |
| 温馨手绘 | 水彩、插画、柔色 | 日常记录、百日 |
| 萌趣卡通 | 动物、气球、糖果色 | 生日、节日 |
| 清新文艺 | 极简、留白、小清新 | 写真、旅行 |
| 复古胶片 | 做旧、暗角、胶片颗粒 | 成长回忆 |
| 节日主题 | 新年、圣诞、中秋等 | 节日纪念 |

**模板内容构成**：

- 封面（含标题、副标题、宝宝姓名、日期）
- 内页（N 种版式：单图大图、双图拼贴、多图九宫格、图文混排等）
- 尾页（寄语、签名位）
- 装饰元素（贴纸、边框、背景图）
- 字体方案（中文+英文各 1 套）
- 配色方案

**功能要求**：

| 项 | 要求 |
| --- | --- |
| 展示形式 | 模板列表瀑布流，每个模板显示缩略图 + 风格标签 |
| 预览 | 点击模板可查看 3-5 张示例内页 |
| 筛选 | 支持按"分类"、"免费/付费"、"热门"筛选 |
| 切换成本 | 选图后切换模板不丢失照片，重新排版即可 |
| 付费策略 | 免费模板 ≥ 5 款，其余可为付费/会员模板 |

---

#### F3：智能自动排版

**功能描述**：根据所选模板和用户上传的照片，自动完成画册排版并生成预览。

**排版规则**：

1. **总页数估算**：根据照片数量和模板默认每页图数（1-4 张）估算页数，控制在 6-30 页。
2. **封面图选择**：优先选择用户标记或系统识别的"高质量、正面、居中"照片。
3. **版式匹配**：根据照片横竖比和数量自动匹配版式（横图走大图版式，竖图走拼贴）。
4. **色彩协调**：自动检测照片主色调，与模板背景做对比度优化（避免主体和背景撞色）。
5. **文字填充**：默认填充"宝宝姓名 + 日期范围 + 模板预设文案"，用户可后期修改。
6. **主体居中**：通过人脸/人物识别，保证照片裁切时不切脸。

**性能要求**：

- 单册生成时长 ≤ 15 秒（20 张图以内）
- 生成失败率 < 1%，失败可重试
- 生成过程提供进度提示

---

#### F4：画册预览与翻阅

**功能描述**：生成完成后，用户可以翻页浏览画册，查看整体效果。

**功能要求**：

| 项 | 要求 |
| --- | --- |
| 翻页方式 | 左右滑动翻页，支持仿真翻书动画 |
| 缩放查看 | 双指缩放查看细节 |
| 页码显示 | 底部显示"当前页/总页数" |
| 快速导航 | 底部缩略图条，点击跳转 |
| 轻编辑 | 支持点击单页替换照片、修改文字（MVP 简化版） |
| 重新生成 | 按钮"换一换"，保留照片换一套排版 |

---

#### F5：画册管理（我的画册）

**功能描述**：用户查看、管理历史生成的画册。

**功能要求**：

| 项 | 要求 |
| --- | --- |
| 列表展示 | 封面缩略图 + 标题 + 创建时间 + 页数 |
| 操作 | 查看、重命名、删除、分享、导出 |
| 排序 | 按创建时间倒序，支持手动置顶 |
| 存储 | 云端存储（账号绑定），本地缓存加速 |

---

#### F6：分享与导出

**功能描述**：用户将生成的画册分享出去或导出保存。

**功能要求**：

| 方式 | 说明 | 是否付费 |
| --- | --- | --- |
| H5 链接分享 | 生成可翻阅的 H5 链接，支持微信/朋友圈分享 | 免费 |
| 长图分享 | 所有页拼接为长图 | 免费 |
| 单页图片保存 | 保存指定页为高清图 | 免费 |
| PDF 导出 | 高清 PDF（300dpi） | 付费/会员 |
| 视频画册 | 自动生成带 BGM 的翻页视频 | 付费/会员 |
| 实体打印（后续） | 邮寄实体画册 | 付费 |

## 五、非功能需求

### 5.1 性能

- 首页加载时间 ≤ 2s
- 模板列表首屏加载 ≤ 1.5s
- 单册生成 ≤ 15s（20 张图以内）
- 画册翻页响应 ≤ 200ms

### 5.2 兼容性

- iOS 13+、Android 8.0+
- 微信小程序（后续）
- 主流浏览器 H5 预览（Chrome/Safari 近两年版本）

### 5.3 安全与隐私

- 照片传输全程 HTTPS
- 照片存储加密，用户删除后 7 天内物理删除
- 未经用户授权不得用于训练 / 商业展示
- 未成年人肖像合规：需监护人同意条款
- 符合《个人信息保护法》《未成年人保护法》相关要求

### 5.4 可用性

- 核心流程操作步数 ≤ 4 步（上传→选模板→生成→预览）
- 新手引导覆盖首次创建流程
- 支持弱网断点续传上传

## 八、风险与应对

| 风险 | 等级 | 应对方案 |
| --- | --- | --- |
| 模板同质化、视觉平庸 | 高 | 与专业插画师/设计师合作，建立模板评审机制 |
| 智能排版效果不及预期（切脸、撞色） | 高 | 引入人脸识别 + 显著性检测；提供"换一换"兜底 |
| 照片隐私担忧 | 高 | 端侧处理优先；明确隐私条款；支持本地模式 |
| 付费转化低 | 中 | 模板免费引流 + 高清导出/视频画册付费；节日营销活动 |
| 生成性能（大图多图） | 中 | 图片端侧压缩 + 服务端异步渲染 + CDN |
